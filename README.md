## Technologies

Site is created with:

- React.js version 17.0.2
- MUI - (for styling)
- Netlify
- React Spring version 9.4.2 (for cool animations!)
- Particles.js
- [Animate.css](https://animate.style)
- [React Waypoint](https://github.com/civiccc/react-waypoint)
