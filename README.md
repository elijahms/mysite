## Technologies

Site is created with:

- React.js version 17.0.2
- MUI - (for styling)
- Netlify
- React Spring version 9.4.2 (for cool animations!)
- Particles.js

## Setup

To run this project, install it locally using npm:

```
$ cd ../mysite
$ npm start --prefix client
$ rails s

```
